# Track-Man

## Simple Pursuit Game in C

![preview](https://github.com/Kesh17/track-man/blob/main/preview.gif)

## Quick Start

### Build (Requires [CMake](https://cmake.org) & [Raylib](https://www.raylib.com/))

```console
$ cmake -Bbuild .
$ cmake --build build
```

### Run

```console
$ ./build/main
```
